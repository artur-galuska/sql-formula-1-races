## Description

This project is an analysis of Formula 1 races and their results via use of SQL queries.
It follows a set of 45 tasks, with increasing difficulty, that must be solved using SQL language on the imported datasets of the races.

The project served as a learning experience and a practical example of how analytics are performed in the real world.
During my work on this project I have learned how to use a new tool - MySQL Workbench - in helping me analyse the datasets.
It provided me with hands-on experience in working with SQL Queries, Subqueries, Joins, Grouping, Filtering, Aggregations, CTEs.

## Usage

The project is provided in .pdf format, both tasks and queries in separate files, as well as all the datasets.<br>
The datasets used for finding the required information are as follows:
- constructors.csv
- drivers.csv
- pitstops.csv
- results.csv

The other 5 datasets were used for general practice and weren't required for the completion of tasks, but are included as a reference of data I had to work with.

_Tasks_for_the_Dataset.pdf_ - provides 45 tasks I had to work through during this project.<br>
_SQL_Queries.pdf_ - provides SQL query code used to find the necessary information.<br>
To see the queries work in practice:<br>
- Download and import the datasets into MySQL Workbench environment (not tested in any other SQL environments)
- Copy and paste the query code from SQL_Queries.pdf into MySQL
- Run the code in MySQL, it should show the results
